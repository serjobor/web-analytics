# web-analytics

## Этапы работы над итоговым проектом
1. Распределиться на команды и записаться в документ. 
2. Сформулировать гипотезы (по 1 на человека из команды). 
3. Определить стек технологий для разработки сайта и начать разработку. 
4. Внедрить счетчик Я.Метрики и проверить его работоспособность.
5. Провести пилотное исследование, собрать и обработать данные, сформулировать результаты.
6. Провести основное исследование, собрать и обработать данные, сформулировать окончательные результаты.
7. Оформить полученные на 6-7 этапах результаты в отчет и на их основе создать презентацию по своей гипотезе. 
8. Выступить с презентацию и загрузить файл с отчетом в Moodle-курс.
